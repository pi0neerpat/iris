## Install

```sh
yarn install
```

copy `.env.template` to `.env` and add the required variables

## Usage

```sh
# Do these commands once
yarn rw db save
yarn rw db up

# Run this to start the app
yarn rw dev
```

### Ngrok it

```bash
# In VM
yarn rw dev --fwd="--host=0.0.0.0"

# on host machine
./ngrok http 8910 --host-header=rewrite
```
